# CS50-week2
In this repository you can find my solutions for Week 2 of CS50. This week was focused on strings, arrays, and character manipulation. It was noticeably more challenging than Week 1 because I had to think about how characters are represented and processed in `C`.
## Content
- [Scrabble](https://github.com/LevGashnikov/CS50-week2/blob/main/scrabble.c) This problem introduced me into arrays and character values. The main challenge was calculating each player's score and ignoring non-alphabetic characters.
- [Readability](https://github.com/LevGashnikov/CS50-week2/blob/main/readability.c) This was my first experience in solving real world problems. The main challenge was not the math part, but the part with taking into account every possible sign.
- [Substitution](https://github.com/LevGashnikov/CS50-week2/blob/main/substitution.c) The hardest problem of the week, the hardest thing was to understand how ASCII values can be used to encrypt text while preserving uppercase and lowercase letters in my mind, and keep everything in my head.
